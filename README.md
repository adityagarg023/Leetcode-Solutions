## Welcome to the **LeetCode Solutions** repository! 🔧
Collection of LeetCode problem solutions implemented in C++. Organized by difficulty (Easy, Medium, Hard) and focused on clarity, efficiency, and optimized problem-solving techniques. Perfect for coding interview preparation and enhancing algorithmic skills.

## 📂 Repository Structure:
- **Easy**: Beginner-friendly problems.
- **Medium**: Intermediate-level challenges.
- **Hard**: Advanced problems for in-depth algorithmic thinking.

## 🛠 How to Use:
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/LeetCode-Solutions.git
   ```
2. Navigate to the folder for the difficulty level or topic:
   ```bash
   cd Medium/
   ```
3. Compile and run the code:
   ```bash
   g++ longest_substring.cpp -o longest_substring
   ./longest_substring
   ```
